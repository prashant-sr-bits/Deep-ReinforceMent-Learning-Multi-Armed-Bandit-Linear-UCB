# Deep-ReinforceMent-Learning-Multi-Armed-Bandit-Linear-UCB
This simulation was carried out complete mandatory assignment work using Multi Armed Bandit using contextualization. 


#Problem Environment 

In the given problem, cancer treatment data set is given. This data set contains prior data which 
captures information as given below :  
1) Characteristics which are related to a Cancer disease.
2) For given Characteristics, what type of Cancer Treatment was taken.  
3) If selected treatment resulted in SUCCESS of FAILURE and cost (money and me) incurred.  

For given problem descrip on and dataset, we have to identify personalized and effective treatment 
strategy.  
To solve this problem, we will apply the concept of Mul-Armed Bandit.
